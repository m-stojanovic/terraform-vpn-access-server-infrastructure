## About The Project

Marketplace AWS Access Server OpenVPN setup with Terraform.

### Built With

* [Terraform](https://terraform.io)
* [OpenVPN](https://openvpn.net)
* [Yaml](https://yaml.com)


## Getting Started

terraform init
terraform apply

### Prerequisites

Terraform version 1.6.5 +
